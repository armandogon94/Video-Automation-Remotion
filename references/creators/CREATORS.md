## How to add a new creator

1. Append a row to the table above (handle, platform, 1-line "why").
2. Scrape — pick the right command for the platform:
   - Instagram reels: `npm run scrape:reels -- --handle <handle> --count 12`
   - YouTube Shorts: `npm run scrape:shorts -- --handle <handle> --count 12` (or `--url <full-channel-shorts-url>`)
3. Extract keyframes for analysis: `npm run analyze:creator -- --handle <handle>`
4. Open the frames, edit `references/creators/<handle>/ANALYSIS.md`:
   - **Identify distinct templates PER VIDEO** — a creator can run many templates. Don't average everything into "their style." For each template, name it + cite which reels use it + describe motion + map to our Stream E typology.

## Analysis rule

> A creator's reels are NOT one style. Treat each reel as a potential distinct template;
> only group reels under the same template if their layout + color + motion are clearly
> shared. This is what carloscuamatzin's ANALYSIS demonstrates — 6 templates, not 1.
