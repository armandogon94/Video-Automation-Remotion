THE RULE THAT GOVERNS EVERYTHING

Claude reads images frame-by-frame, not as motion.

Every prompt that asks Claude to recreate an animation MUST include a transitionVerb sentence in imperative voice. Style transfers from a reference image; choreography does not. We baked this rule into every new Wave-5 schema's transitionVerb field.

I find that sometimes it's really good at interpreting things like styles, but not necessarily at the animations or how you'd like the animation to happen. So it's kind of good initially to give Claude some context as to or as much context as possible about the animation. Just because it cannot really analyze more than frame by frame, right? So it doesn't necessarily understand the transitions. — Tella, ykBDqicGx0M.txt L143–148

BATCH A · 3 NEW ANIMATION PRIMITIVES

Cross-cutting motion utilities · no UI baked in

All in src/animation/. Pure functions, no React, fully tested by tsc.

pathDraw.ts

SVG stroke-dashoffset animator with easing + direction modes.

  • Direction: start-to-end · end-to-start · center-out
  • Easing: outCubic / outQuart / inOutCubic / linear
  • Bonus: pathDrawStaggered for sequential edges (decision-tree, mesh)
  • Soft opacity ramp prevents hairline tail-flash
heldStagger.ts

Per-index reveal with explicit dwell beats (different from continuous stagger).

  • Returns {phase, progress, isHeld} per item
  • Hormozi/Klaus listicle pattern
  • Notification cascade pattern
  • "Punctuate one tier at a time" choreography
smartZoom.ts

Ken Burns + punch-in zoom for video / image content.

  • kenBurns() — gentle 1.0→1.08 over ~4s, inOutCubic
  • punchIn() — hard 1.0→1.15 over 4–6f, held
  • Focal-point anchoring (no zoom-drift)
  • Tella spec: harsh punch-in lands on the emotion peak

BATCH C · 5 HORMOZI TYPOGRAPHY OVERLAYS

One file · HormoziOverlays.tsx · pure typography

The red-team finding pivoted Hormozi's contribution from "complex motion graphics" to "5 disciplined typography overlays sitting on B-roll." All 5 in one file.

<PersistentHookPill>

The thumb-stop: white rounded pill at top, bold black sans, NO entrance animation by default. Present from frame 0 to end.

6 of 10 Hormozi Shorts use this exact pattern.

<ChapterCardPill>

Mid-frame white card with optional number prefix. Crossfade-in, hold, crossfade-out. The beat-by-beat marker.

<ChapterCardDoubleLine>

No pill. Number + bold sans top line + italic-quoted serif bottom line ("2# Prescribe \"the best\""). Drop-shadow for B-roll legibility.

<OpeningTitleCard> · <HookEmojiStrip>

OpeningTitleCard — one-shot 60-frame opener (eyebrow + hero + dek) with scale-pop.

HookEmojiStrip — 2-4 emojis above the hook (📈❌🍀 semantic signaling). Distinctive Hormozi pattern Tella didn't surface.
