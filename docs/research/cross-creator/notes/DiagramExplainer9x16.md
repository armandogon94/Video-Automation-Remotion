Our DiagramExplainer is itself directly modeled on @carloscuamatzin's cream-flowchart
"EL FLUJO" reel (see comp header comment) — a vertical stacked flow-chart, which is the
natural concept-diagram realization of the midu discipline in our own brand.

**What matches (strong):**
- ONE-ACCENT discipline is textbook. Sampled RGB: every structural element — top kicker
  bar ("CLAUDE CODE · SUBAGENTS"), eyebrow ("EL FLUJO"), all card borders, the connecting
  arrows (161,61,61 warm-red), and the mono meta labels (muted warm-red 187,112,119) — is
  the SAME warm-red family. Titles are pure near-black, paper is cream. No second accent
  anywhere. This is exactly midu's gold-only restraint, in our accent.
- Monospace meta text (`$ you`, `auto · proactive`, `Read · Grep · Glob`) = midu's
  monospace/code register. Subtle, lowercase, accent-tinted — disciplined.
- Structured card surface: white rounded cards, thin 2px accent border, soft shadow —
  the "code editor panel" feel translated to a node card.
- Single subject-tool accent: accentColor derives from subjectTool (Claude Code → warm
  clay/red via getToolAccentForSurface), so the accent is content-driven, never arbitrary.

**Motion timing (across our 4 frames):** Sequential top-to-bottom reveal — frame 1 shows
node 1 only, frame 2 adds node 2 mid-spring, frame 3 has both nodes + the connecting arrow
drawn, frame 4 adds a ghosted 3rd node + second arrow. Cards enter on the critically-damped
"Bloomberg" spring (22/130/0.7); arrows draw-on 0.2s after their target node lands, with a
late-painting arrowhead. This staged "build the diagram one box at a time" beat is the
correct concept-diagram motion grammar.

**What differs (correct / expected):** Copy is our own Spanish ("Escribes una función",
"Claude invoca al subagente"); midu's is JS/SVG code. Our surface is cream-light vs midu's
dark IDE — but the comp supports a `dark` palette mode (#10172A cards, amber shadow) for
exact tonal parity when desired. Differing copy + surface tone is expected for a generic
template; the discipline transfers cleanly.
